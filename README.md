# 🚀 Log to Staging Automation

> Automating repetitive support workflows with a “work smart, not hard” mindset.

---

## 📖 Overview

This project was built to **automate my daily operational tasks** as part of a Technical Support workflow.

It is a **fully custom-built PowerShell solution**, not an official tool, designed to solve real-world challenges encountered during day-to-day work.

Instead of manually:
- copying files from network shares  
- extracting multiple layers of compressed files  
- organizing folders  
- and checking staging status  

This script automates the entire process end-to-end.

> 💡 **Work smarter, not harder — reduce manual effort and eliminate repetitive tasks.**

---

## 🛠️ Custom Solution

This is a **completely customized script built from scratch** to support my workflow.

It is **not provided by Commvault**, but developed independently to:
- Address real operational challenges  
- Improve efficiency during escalations  
- Reduce reliance on manual processes  

---

## 🎯 Purpose

This tool focuses on:

- ⚡ Minimizing manual intervention  
- 🔁 Eliminating repetitive tasks  
- 🧠 Improving efficiency in daily work  
- ⏱️ Saving time during escalation handling  

It transforms a multi-step manual process into a **single automated workflow**.

---

## ⚙️ Features

- 📂 Copy files from network shares  
- 📦 Extract `.zip` and `.7z` archives  
- 🔄 Recursive extraction (handles nested archives)  
- 🗂️ Organize output into structured folders  
- 🧾 Track processed CCID & Ticket data  
- 🗃️ Prepare database files for staging  
- 🔐 Authenticate with Dataverse (CRM) 
- 🚀 Submit staging requests using extracted DB data
